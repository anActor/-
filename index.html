项目概述

ValoPlant 是什么？

一个专业的 Valorant 游戏战术工具平台，为玩家和战队提供地图策略规划、阵容分析、战术学习等功能。

目标用户

🎨 绘制工具

支持画笔、箭头、标记点、文本注释等多种绘制工具

👥 实时协作

多人同时编辑，实时同步战术变更

💾 保存分享

战术方案保存和团队间分享功能

🗺️ 地图管理

支持攻防双方视角，多层级地图管理

技术特点： 基于 Canvas/WebGL 或 SVG 实现，支持复杂图形绘制和实时协作编辑

核心功能 2: 阵容管理系统

📊 技能数据库

完整的英雄技能信息和数据分析

🤖 阵容推荐

基于算法的智能阵容搭配建议

📈 对战分析

历史对战记录分析和数据统计

📚 教程系统

1-10 序列化教程，系统学习战术知识

技术架构

  • React.js + TypeScript

  • Ant Design / Material-UI
  • Redux Toolkit 状态管理

⚙️ 后端技术

  • Node.js + Express
  • PostgreSQL + Redis
  • JWT + OAuth2.0
  • AWS S3 文件存储
